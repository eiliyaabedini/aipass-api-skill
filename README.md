# AI Pass API Skill

> Complete AI Pass API skill for AI agents — text generation, image generation, image editing, TTS, transcription, and video generation.

## Install

```bash
npx skills add eiliyaabedini/aipass-api-skill
```

## What's included

| Feature | Models |
|---------|--------|
| 💬 Text Generation | GPT-5, Claude Opus/Sonnet/Haiku, Gemini 2.5/3 |
| 🎨 Image Generation | Flux Pro, DALL-E 3, GPT-Image-1, Imagen 4 |
| ✏️ Image Editing | Gemini 3 Pro Image |
| 🔊 Text-to-Speech | TTS-1, TTS-1-HD, GPT-4o Mini TTS |
| 🎙️ Transcription | Whisper-1 |
| 🎬 Video Generation | Veo 3.1, Sora 2 |
| 🔢 Embeddings | text-embedding-3-small/large |

## Setup

1. Get your API key: [aipass.one/panel/developer.html](https://aipass.one/panel/developer.html)
2. Set env var: `export AIPASS_API_KEY=your_key_here`

## About AI Pass

[AI Pass](https://aipass.one) is your universal AI wallet — one key, all AI models, pay as you go. Developers earn **50% commission** on every API call their users make.
